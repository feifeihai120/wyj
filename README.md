Ionic SJ App
============

```bash
$ npm install -g ionic
$ npm install
$ bower install

$ ionic platform add ios
$ ionic platform add android

$ ionic build ios
$ ionic emulate ios

$ ionic run android

$ ionic serve

$ grunt serve
```
